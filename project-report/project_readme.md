# Adaptive Learning Management System (ALMS)

## Overview
This project implements an Adaptive Learning Management System that personalizes learning based on student preferences and progress.

## Key Features
- User login and dashboard
- Course and module management
- Feedback and adaptive content flow
- Django-style backend and static frontend
- Database with ERD and DFD
